Updates in Homework #4 (branch hw4-validation)
In this homework, the following changes were made:

2. Removal of the ctrlWrapper function: With the upgrade to Express 5, error handling has become more streamlined, making the ctrlWrapper function unnecessary. As a result, this wrapper has been removed.

3. Changes in Routes: Consequently, all routes have been updated to remove the ctrlWrapper, resulting in cleaner and more maintainable code.

These updates simplify the error-handling logic and transition the project to a more modern framework version.
